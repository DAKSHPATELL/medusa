# ClearBorder Demo — Golden Path

This directory contains the scripted demo flow for the ClearBorder presentation.

## Structure
- `golden-path.ts` — End-to-end scripted demo (Phase 5)
- `seed-data.json` — Seeded case data for the demo
- `seed-call.*` — Recorded audio clips for Live Translate demo

## Running
```bash
CASE_STORE=local DEMO_MODE=true npm run demo
```
